# Функции

## Определение функции

<LessonSection title="Определение функции">

**Функцией** называется правило, которое каждому элементу $x$ из множества $X$ ставит в соответствие единственный элемент $y$ из множества $Y$.

Обозначается: $f: X \to Y$ или $y = f(x)$

**Компоненты функции:**
- $x$ — аргумент (независимая переменная)
- $y$ — значение функции (зависимая переменная)
- $f$ — правило соответствия

</LessonSection>

## График функции

<LessonSection title="График функции">

**Графиком функции** $y = f(x)$ называется множество всех точек плоскости с координатами $(x, f(x))$.

**Основное свойство (вертикальный тест):** Любая вертикальная прямая пересекает график функции **не более чем в одной точке**.

**Пример графика, который является функцией:**
![График функции](https://i.postimg.cc/NM7K2s4N/изображение.png)

**Пример графика, который НЕ является функцией:**
![Не функция](https://i.postimg.cc/PxFB4Z2D/изображение.png)

</LessonSection>

## Работа со значениями функций

<LessonSection title="Определение значения функции в точке">

Чтобы найти значение функции $f(a)$:

1. Находим точку $a$ на оси OX
2. Проводим вертикальную прямую до пересечения с графиком
3. От точки пересечения проводим горизонтальную прямую к оси OY
4. Координата на оси OY — значение функции

**Пример на графике:**
![Определение значения функции](https://i.postimg.cc/Wp043132/изображение.png)

$f(-4) = 2$
$f(7) = -3$

</LessonSection>

<LessonSection title="Нахождение аргумента по значению функции">

Чтобы найти $x$, при котором $f(x) = c$:

1. Находим точку $c$ на оси OY
2. Проводим горизонтальную прямую до пересечения с графиком
3. От точек пересечения проводим вертикальные прямые к оси OX
4. Координаты на оси OX — искомые значения аргумента

**Пример на графике:**
![Определение значения функции](https://i.postimg.cc/Wp043132/изображение.png)
$f(x) = 3$ при $x = -7, -5, -2, 1$

</LessonSection>

## Область определения и множество значений

<LessonSection title="Область определения и множество значений">

**Область определения** $D(f)$ — множество всех значений аргумента $x$, при которых функция определена.

**Множество значений** $E(f)$ — множество всех значений, которые принимает функция.

![Область определения и множество значений](https://i.postimg.cc/CKJyR988/изображение.png)

**Пример:** $D(f) = [1, 6]$, $E(f) = [1, 5]$

</LessonSection>

<LessonSection title="Примеры нахождения области определения">

**а)** $f(x) = \sqrt{x^2 - 5x + 4}$

$$
\begin{aligned}
x^2 - 5x + 4 &\geq 0 \\
(x-1)(x-4) &\geq 0 \\
x &\in (-\infty, 1] \cup [4, +\infty)
\end{aligned}
$$

**б)** $h(x) = \dfrac{5}{\sqrt{x-4}} + \sqrt{13-2x}$

$$
\begin{cases}
x - 4 > 0 \\
13 - 2x \geq 0
\end{cases}
$$

$$
\begin{cases}
x > 4 \\
x \leq 6{,}5
\end{cases}
$$

$D(h) = (4; 6{,}5]$

</LessonSection>

<LessonSection title="Примеры нахождения множества значений">

**а)** $f(x) = |x| + 4$

Поскольку $|x| \geq 0$, то $|x| + 4 \geq 4$

$E(f) = [4, +\infty)$

**б)** $g(x) = -x^2 + 6x - 2$

Найдем вершину параболы:

$$
\begin{aligned}
x_{\text{в}} &= -\frac{b}{2a} = -\frac{6}{-2} = 3 \\
y_{\text{в}} &= -9 + 18 - 2 = 7
\end{aligned}
$$

Так как $a = -1 < 0$ (ветви вниз):

$E(g) = (-\infty, 7]$

</LessonSection>

## Нули функции

<LessonSection title="Нули функции">

**Нули функции** — это значения аргумента $x$, при которых значение функции равно нулю.

Математически: $f(x_0) = 0$, где $x_0$ — нуль функции.

**Геометрический смысл:** Нули функции — это абсциссы точек пересечения графика функции с осью OX.

![Нули функции](https://i.postimg.cc/66z1XdXD/изображение.png)

**Пример:** $f(x) = 0$ при $x = -2, 4, 8$

</LessonSection>

## Промежутки знакопостоянства

<LessonSection title="Промежутки знакопостоянства">

**Промежутками знакопостоянства** называются интервалы значений аргумента, на которых функция сохраняет постоянный знак (положительный или отрицательный).

![Промежутки знакопостоянства](https://i.postimg.cc/HshZJdjQ/изображение.png)

**Функция положительна** $(f(x) > 0)$:
Там, где график находится **выше** оси OX.
$x \in (-2, 4) \cup (8, 9)$


**Функция отрицательна** $(f(x) < 0)$:
Там, где график находится **ниже** оси OX.

$x \in (-9, -2) \cup (4, 8)$

</LessonSection>

## Монотонность функции

<LessonSection title="Монотонность функции">

**Возрастающая функция:** Функция $f(x)$ называется возрастающей на промежутке, если для любых $x_1 < x_2$ из этого промежутка выполняется $f(x_1) < f(x_2)$.

![Возрастающая функция](https://i.postimg.cc/DZvdjM80/изображение.png)

**Убывающая функция:** Функция $f(x)$ называется убывающей на промежутке, если для любых $x_1 < x_2$ из этого промежутка выполняется $f(x_1) > f(x_2)$.

![Убывающая функция](https://i.postimg.cc/T1hjG5VF/изображение.png)

</LessonSection>

<LessonSection title="Пример анализа монотонности">

![Пример монотонности](https://i.postimg.cc/59kqgp1y/изображение.png)

**Функция возрастает:**
$x \in (-9, -8) \cup (-7.5, 1.5) \cup (6, 9)$

**Функция убывает:**
$x \in (-8, -6.5) \cup (1.5, 6)$

**Важные точки:**
- **Точки максимума:** $x = -8$ и $x = 1.5$ (переход от возрастания к убыванию)
- **Точки минимума:** $x = -6.5$ и $x = 6$ (переход от убывания к возрастанию)

</LessonSection>

<LessonSection title="Анализ конкретных функций">

**а)** $f(x) = 6 - 1.5x$

- **Нули функции:** $6 - 1.5x = 0 \Rightarrow x = 4$
- **Знакопостоянство:** $f(x) > 0$ при $x < 4$; $f(x) < 0$ при $x > 4$
- **Монотонность:** Коэффициент $-1.5 < 0$, функция **убывает** на $(-\infty, +\infty)$

**б)** $g(x) = x^2 - 4x + 3$

- **Нули функции:** $D = 4$, $x_1 = 1$, $x_2 = 3$
- **Знакопостоянство:** $g(x) > 0$ при $x \in (-\infty, 1) \cup (3, +\infty)$; $g(x) < 0$ при $x \in (1, 3)$
- **Монотонность:** Вершина $x_{\text{в}} = 2$. **Убывает** на $(-\infty, 2]$, **возрастает** на $[2, +\infty)$

**в)** $h(x) = |x| + 5$

- **Нули функции:** Нет, так как $|x| + 5 > 0$ для всех $x$
- **Знакопостоянство:** $h(x) > 0$ для всех $x$
- **Монотонность:** **Убывает** на $(-\infty, 0]$, **возрастает** на $[0, +\infty)$

</LessonSection>

## Четная функция

<LessonSection title="Определение четной функции">

**Функция $y = f(x)$ называется четной**, если для любого $x$ из области определения выполняется:

1. $x \in D(f) \Rightarrow -x \in D(f)$
2. $f(-x) = f(x)$

**Геометрический смысл:** График четной функции симметричен относительно оси OY.

![Симметрия четной функции](https://i.postimg.cc/TwQSCsqB/изображение.png)

Если точка $(x, y)$ принадлежит графику четной функции, то и точка $(-x, y)$ также принадлежит графику.

</LessonSection>

<LessonSection title="Примеры четных функций">

![Четная функция - парабола](https://i.postimg.cc/kGpHdtZ3/изображение.png)

![Четная функция - модуль](https://i.postimg.cc/7PsdzpLK/изображение.png)

![Четная функция - косинус](https://i.postimg.cc/NfKVkp9B/изображение.png)

**Примеры:** $y = x^2$, $y = |x|$, $y = \cos x$

</LessonSection>

## Нечетная функция

<LessonSection title="Определение нечетной функции">

**Функция $y = f(x)$ называется нечетной**, если для любого $x$ из области определения выполняется:

1. $x \in D(f) \Rightarrow -x \in D(f)$
2. $f(-x) = -f(x)$

**Геометрический смысл:** График нечетной функции симметричен относительно начала координат.

![Симметрия нечетной функции](https://i.postimg.cc/TYm8XtRt/изображение.png)

Если точка $(x, y)$ принадлежит графику нечетной функции, то и точка $(-x, -y)$ также принадлежит графику.

</LessonSection>

<LessonSection title="Примеры нечетных функций">

![Нечетная функция - кубическая парабола](https://i.postimg.cc/9QtvDzwC/изображение.png)

![Нечетная функция - синус](https://i.postimg.cc/W4Nxgj0H/изображение.png)

![Нечетная функция - гипербола](https://i.postimg.cc/KzVW6MwH/изображение.png)

**Примеры:** $y = x^3$, $y = \sin x$, $y = \dfrac{1}{x}$

</LessonSection>

<LessonSection title="Алгоритм определения четности функции">

1. **Проверить область определения:** Должна быть симметрична относительно нуля (если $x \in D(f)$, то $-x \in D(f)$)
2. **Вычислить $f(-x)$:** Упростить выражение
3. **Сравнить с $f(x)$:**
   - Если $f(-x) = f(x)$ — функция **четная**
   - Если $f(-x) = -f(x)$ — функция **нечетная**
   - Если оба условия не выполняются — функция **ни четная, ни нечетная**

</LessonSection>

<LessonSection title="Примеры определения четности">

**а)** $f(x) = 7x^3$
- $D(f) = \mathbb{R}$ — симметрична
- $f(-x) = 7(-x)^3 = -7x^3 = -f(x)$
- **Ответ:** функция нечетная

**б)** $d(x) = -6x^4 - 8$
- $D(d) = \mathbb{R}$ — симметрична
- $d(-x) = -6(-x)^4 - 8 = -6x^4 - 8 = d(x)$
- **Ответ:** функция четная

**в)** $h(x) = -\sqrt{2x}$
- $D(h) = [0, +\infty)$ — НЕ симметрична
- **Ответ:** функция ни четная, ни нечетная

</LessonSection>

<LessonSection title="Использование свойств четности">

**Задача 1 (четная функция):** Известно, что $f(x)$ — четная, $f(3) = -7$, $f(-4) = 5$. Найти $2f(-3) - f(4)$.

**Решение:**
Для четной функции: $f(-x) = f(x)$

1. $f(-3) = f(3) = -7$
2. $f(4) = f(-4) = 5$
3. $2f(-3) - f(4) = 2 \cdot (-7) - 5 = -19$

**Ответ:** $-19$

**Задача 2 (нечетная функция):** Известно, что $f(x)$ — нечетная, $f(-5) = 3$, $f(2) = -8$. Найти $4f(5) + f(-2)$.

**Решение:**
Для нечетной функции: $f(-x) = -f(x)$

1. $f(5) = -f(-5) = -3$
2. $f(-2) = -f(2) = -(-8) = 8$
3. $4f(5) + f(-2) = 4 \cdot (-3) + 8 = -4$

**Ответ:** $-4$

</LessonSection>

## Преобразования графиков

<LessonSection title="Преобразования графиков функций">

Графики функций можно перемещать (сдвигать) вдоль осей координат. Эти преобразования сохраняют форму графика, меняя только его положение.

**Основные виды преобразований:**
1. Сдвиг вверх/вниз (вдоль оси OY)
2. Сдвиг влево/вправо (вдоль оси OX)

</LessonSection>

<LessonSection title="Сдвиг вдоль оси OY (вверх/вниз)">

**Общее правило:** График функции $y = f(x) + b$ получается из графика $y = f(x)$ сдвигом на $b$ единиц вдоль оси OY.

- Если $b > 0$ — сдвиг **вверх**
![Сдвиг вверх](https://i.postimg.cc/85YT5sJB/изображение.png)
- Если $b < 0$ — сдвиг **вниз**
![Сдвиг вниз](https://i.postimg.cc/Bb141s2j/изображение.png)

**Примеры:**
- $y = \sqrt{x} + 4$ — сдвиг на 4 единицы вверх
![Сдвиг вверх](https://i.postimg.cc/Jz37q1vk/изображение.png)
- $y = \sqrt{x} - 4$ — сдвиг на 4 единицы вниз
![Сдвиг вверх](https://i.postimg.cc/YSprD5wK/изображение.png)



</LessonSection>

<LessonSection title="Сдвиг вдоль оси OX (влево/вправо)">

**Общее правило:** График функции $y = f(x - a)$ получается из графика $y = f(x)$ сдвигом на $a$ единиц вдоль оси OX.

- $y = f(x - a)$ — сдвиг на $a$ единиц **вправо**
![Сдвиг вправо](https://i.postimg.cc/BvYjTSP2/изображение.png)
- $y = f(x + a)$ — сдвиг на $a$ единиц **влево**
![Сдвиг влево](https://i.postimg.cc/rmDKgBS8/изображение.png)


**Важно запомнить:** Знак в скобке противоположен направлению сдвига!

**Примеры:**
- $y = 2(x - 6)^2$ — сдвиг на 6 единиц вправо
![Сдвиг вправо](https://i.postimg.cc/kgCR9hd6/изображение.png)
- $y = \dfrac{1}{x + 2}$ — сдвиг на 2 единицы влево
![Сдвиг влево](https://i.postimg.cc/mgSzrNbv/изображение.png)



</LessonSection>

<LessonSection title="Примеры построения графиков">

**Построить графики с помощью преобразований функции $y = |x|$:**

**а)** $y = |x + 4|$
Преобразование: $f(x + 4)$ — сдвиг на 4 единицы **влево**

![График y=|x+4|](https://i.postimg.cc/ZY3qRFHF/изображение.png)

**б)** $y = |x| - 4$
Преобразование: $f(x) - 4$ — сдвиг на 4 единицы **вниз**

![График y=|x|-4](https://i.postimg.cc/gkW26XtC/изображение.png)

**в)** $y = |x - 3| + 2$
Два преобразования:
1. $|x - 3|$ — сдвиг на 3 единицы вправо
2. $|x - 3| + 2$ — сдвиг на 2 единицы вверх

![График y=|x-3|+2](https://i.postimg.cc/Nfkf8sW2/изображение.png)

</LessonSection>

<LessonSection title="Алгоритм построения графиков с помощью преобразований">

1. **Определить исходную функцию** (базовый график)
2. **Проанализировать преобразования:**
   - Если $f(x \pm a)$ — сдвиг вдоль OX
   - Если $f(x) \pm a$ — сдвиг вдоль OY
3. **Определить направление сдвигов:**
   - $f(x + a)$ — сдвиг влево на $a$
   - $f(x - a)$ — сдвиг вправо на $a$
   - $f(x) + a$ — сдвиг вверх на $a$
   - $f(x) - a$ — сдвиг вниз на $a$
4. **Выполнить преобразования последовательно**
5. **Построить график**

**Важно:** При нескольких преобразованиях порядок выполнения иногда важен. Сначала выполняют сдвиги вдоль OX, затем вдоль OY.

</LessonSection>

<LessonSection title="Примеры сложных преобразований">

**Пример 1:** $y = \sqrt{x - 2} + 3$

Преобразования для $y = \sqrt{x}$:
1. $y = \sqrt{x - 2}$ — сдвиг на 2 единицы вправо
2. $y = \sqrt{x - 2} + 3$ — сдвиг на 3 единицы вверх

**Пример 2:** $y = |x + 1| - 2$

Преобразования для $y = |x|$:
1. $y = |x + 1|$ — сдвиг на 1 единицу влево
2. $y = |x + 1| - 2$ — сдвиг на 2 единицы вниз

</LessonSection>

# Периодические функции

<LessonSection title="Периодические функции">

**Определение:** Функция $y = f(x)$ называется **периодической**, если существует число $T \neq 0$ (период), такое что для любого $x$ из области определения:
1. $x + T \in D(f)$
2. $f(x + T) = f(x)$

**Наименьший положительный период** — это наименьшее положительное $T$, удовлетворяющее условию.

**Смысл:** График повторяется через равные промежутки $T$.

</LessonSection>

<LessonSection title="Пример построения периодического графика">

**Дано:** Функция с периодом $T = 11$ задана на $[-6, 5]$.

![Один период](https://i.postimg.cc/rFf7xtyz/изображение.png)

**Задача:** Достроить график на два соседних периода.


**Длина периода:** $5 - (-6) = 11 = T$

**Период слева** $[-17, -6]$:  
Сдвигаем график с $[-6, 5]$ на 11 единиц влево: $f(x) = f(x + 11)$

**Период справа** $[5, 16]$:  
Сдвигаем график с $[-6, 5]$ на 11 единиц вправо: $f(x) = f(x - 11)$

**Итог:**  
![Три периода](https://i.postimg.cc/8c6sMVMk/изображение.png)



1. Найти длину данного промежутка — это период $T$
2. Для построения **периода слева**: сдвинуть график на $T$ единиц влево
3. Для построения **периода справа**: сдвинуть график на $T$ единиц вправо
4. Повторять при необходимости

</LessonSection>
