# Дробно-рациональные уравнения

## Основное свойство: равенство дроби нулю

<LessonSection title="Основное свойство: равенство дроби нулю">

**Дробь равна нулю тогда и только тогда, когда её числитель равен нулю, а знаменатель не равен нулю.**

$$
\frac{P(x)}{Q(x)} = 0 \quad \Leftrightarrow \quad 
\begin{cases}
P(x) = 0 \\
Q(x) \neq 0
\end{cases}
$$

Это ключевое свойство, которое позволяет решать дробно-рациональные уравнения.

</LessonSection>

## Пример 1: Простое дробно-рациональное уравнение

<LessonSection title="Пример 1: Простое дробно-рациональное уравнение">

Решите уравнение: $\dfrac{x-6}{x+6} = 0$

**Решение:**
$$
\frac{x-6}{x+6} = 0 \quad \Leftrightarrow \quad 
\begin{cases}
x - 6 = 0 \\
x + 6 \neq 0
\end{cases}
\quad \Leftrightarrow \quad 
\begin{cases}
x = 6 \\
x \neq -6
\end{cases}
\quad \Leftrightarrow \quad x = 6
$$

**Ответ:** $x = 6$

</LessonSection>

## Пример 2: Уравнение с разложением на множители

<LessonSection title="Пример 2: Уравнение с разложением на множители">

Решите уравнение: $\dfrac{x^2 - 36}{x - 6} = 0$

**Решение:**
$$
\frac{x^2 - 36}{x - 6} = 0 \quad \Leftrightarrow \quad 
\begin{cases}
x^2 - 36 = 0 \\
x - 6 \neq 0
\end{cases}
$$

Разложим числитель на множители:
$$x^2 - 36 = 0 \quad \Leftrightarrow \quad (x-6)(x+6) = 0 \quad \Leftrightarrow \quad x = 6 \text{ или } x = -6$$

Учитываем ограничение: $x - 6 \neq 0 \Rightarrow x \neq 6$

Из двух корней оставляем только тот, который удовлетворяет ограничению: $x = -6$

**Ответ:** $x = -6$

</LessonSection>

## Пример 3: Уравнение с двумя корнями

<LessonSection title="Пример 3: Уравнение с двумя корнями">

Решите уравнение: $\dfrac{x^2 - 6x}{x + 6} = 0$

**Решение:**
$$
\frac{x^2 - 6x}{x + 6} = 0 \quad \Leftrightarrow \quad 
\begin{cases}
x^2 - 6x = 0 \\
x + 6 \neq 0
\end{cases}
$$

Решаем уравнение числителя:
$$x^2 - 6x = 0 \quad \Leftrightarrow \quad x(x-6) = 0 \quad \Leftrightarrow \quad x = 0 \text{ или } x = 6$$

Проверяем ограничение: $x + 6 \neq 0 \Rightarrow x \neq -6$

Оба корня удовлетворяют ограничению (ни 0, ни 6 не равны -6).

**Ответ:** $x = 0$, $x = 6$

</LessonSection>

## Пример 4: Уравнение без решений

<LessonSection title="Пример 4: Уравнение без решений">

Решите уравнение: $\dfrac{x + 6}{x^2 - 36} = 0$

**Решение:**
$$
\frac{x + 6}{x^2 - 36} = 0 \quad \Leftrightarrow \quad 
\begin{cases}
x + 6 = 0 \\
x^2 - 36 \neq 0
\end{cases}
$$

Решаем уравнение числителя: $x + 6 = 0 \Rightarrow x = -6$

Решаем ограничение:
$$x^2 - 36 \neq 0 \quad \Leftrightarrow \quad (x-6)(x+6) \neq 0 \quad \Leftrightarrow \quad x \neq 6 \text{ и } x \neq -6$$

Корень числителя $x = -6$ НЕ удовлетворяет ограничению, так как он обращает знаменатель в ноль.

**Ответ:** Нет корней

</LessonSection>

# Дробно-рациональные уравнения

<LessonSection title="Пример 5: Уравнение с несколькими дробями">

Решите уравнение: $\dfrac{x}{x^2 - 5x + 6} + \dfrac{3}{x-2} = \dfrac{2}{x-3}$

**Шаг 1: Найдём ОДЗ**
Для каждой дроби знаменатель не должен быть равен нулю:
$$
\begin{cases}
x^2 - 5x + 6 \neq 0 \\
x - 2 \neq 0 \\
x - 3 \neq 0
\end{cases}
$$

Решаем: $x^2 - 5x + 6 = 0$

$$D = 25 - 24 = 1$$

$$x_1 = 2$$

$$x_2 = 3$$

Значит: $x^2 - 5x + 6 = (x-2)(x-3) \neq 0 \Rightarrow x \neq 2$ и $x \neq 3$

Объединяя все условия: $x \neq 2$ и $x \neq 3$

**Шаг 2: Приведём всё к одной дроби**
Перенесем все члены в левую часть:
$$\dfrac{x}{x^2 - 5x + 6} + \dfrac{3}{x-2} - \dfrac{2}{x-3} = 0$$

Заменим $x^2 - 5x + 6 = (x-2)(x-3)$:

$$\dfrac{x}{(x-2)(x-3)} + \dfrac{3}{x-2} - \dfrac{2}{x-3} = 0$$

Приведём все дроби к общему знаменателю $(x-2)(x-3)$:

$$\dfrac{x}{(x-2)(x-3)} + \dfrac{3(x-3)}{(x-2)(x-3)} - \dfrac{2(x-2)}{(x-2)(x-3)} = 0$$

Объединим в одну дробь:

$$\dfrac{x + 3(x-3) - 2(x-2)}{(x-2)(x-3)} = 0$$

**Шаг 3: Используем основное свойство дроби**
Дробь равна нулю тогда и только тогда, когда числитель равен нулю, а знаменатель не равен нулю:
$$
\begin{cases}
x + 3(x-3) - 2(x-2) = 0 \\
(x-2)(x-3) \neq 0
\end{cases}
$$

**Шаг 4: Решим уравнение числителя**
$$x + 3x - 9 - 2x + 4 = 0$$

$$2x - 5 = 0$$

$$x = \dfrac{5}{2} = 2.5$$

**Шаг 5: Проверим ОДЗ**
1. $x = 2.5 \neq 2$ — верно
2. $x = 2.5 \neq 3$ — верно
3. Знаменатель $(x-2)(x-3)$ при $x = 2.5$ равен

 $0.5 \times (-0.5) = -0.25 \neq 0$

Корень удовлетворяет всем условиям ОДЗ.

**Шаг 6: Проверка подстановкой (по желанию)**
Подставим $x = 2.5$ в исходное уравнение:
$$\dfrac{2.5}{6.25 - 12.5 + 6} + \dfrac{3}{0.5} = \dfrac{2}{-0.5}$$
$$\dfrac{2.5}{-0.25} + 6 = -4$$
$$-10 + 6 = -4$$
$$-4 = -4 \quad \text{✓}$$

**Ответ:** $x = 2.5$ или $x = \dfrac{5}{2}$

**Примечание:** Этот метод приведения всех членов к одной дроби часто более удобен, так как:
1. Мы сразу получаем дробь, равную нулю
2. Легко применяем основное свойство дроби
3. Меньше вероятность арифметических ошибок при переносе слагаемых

</LessonSection>

<LessonSection title="Пример 6: Нахождение нулей функции">

Найдите нули функции: $f(x) = \dfrac{x^3 - 7x^2 + 12x}{x-4}$

**Шаг 1: Найдём ОДЗ**
$$x - 4 \neq 0 \Rightarrow x \neq 4$$

**Шаг 2: Найдём нули функции**
$$f(x) = 0 \Leftrightarrow \dfrac{x^3 - 7x^2 + 12x}{x-4} = 0$$

Разложим числитель на множители:
$$x^3 - 7x^2 + 12x = x(x^2 - 7x + 12)$$
$$= x(x-3)(x-4)$$

Уравнение: $x(x-3)(x-4) = 0$

Корни: $x = 0$, $x = 3$, $x = 4$

**Шаг 3: Проверим на принадлежность ОДЗ**
Из ОДЗ: $x \neq 4$, поэтому $x = 4$ не входит в область определения.

**Ответ:** Нули функции: $x = 0$ и $x = 3$

</LessonSection>

<LessonSection title="Пример 7: Уравнение, решаемое методом замены переменной">

Решите уравнение: $\dfrac{2x-1}{x} + \dfrac{x}{2x-1} = 4.25$

**Шаг 1: Найдём ОДЗ**
$$
\begin{cases}
x \neq 0 \\
2x - 1 \neq 0 \Rightarrow x \neq 0.5
\end{cases}
$$

**Шаг 2: Сделаем замену переменной**
Пусть $t = \dfrac{2x-1}{x}$, тогда $\dfrac{x}{2x-1} = \dfrac{1}{t}$

Преобразуем 4.25: $4.25 = \dfrac{17}{4}$

Уравнение принимает вид: $t + \dfrac{1}{t} = \dfrac{17}{4}$

**Шаг 3: Решим полученное уравнение**

Умножим на 4t: $4t^2 + 4 = 17t$

$$4t^2 - 17t + 4 = 0$$

$$D = 289 - 64 = 225$$

$$\sqrt{D} = 15$$

$$t_1 = \dfrac{17-15}{8} = \dfrac{1}{4}$$
$$t_2 = \dfrac{17+15}{8} = 4$$

**Шаг 4: Вернёмся к переменной x**

**Случай 1:** $t = \dfrac{1}{4}$

$$\dfrac{2x-1}{x} = \dfrac{1}{4}$$

$$4(2x-1) = x$$

$$8x - 4 = x$$

$$7x = 4$$

$$x = \dfrac{4}{7}$$

**Случай 2:** $t = 4$

$$\dfrac{2x-1}{x} = 4$$

$$2x - 1 = 4x$$

$$-2x = 1$$

$$x = -\dfrac{1}{2}$$

**Шаг 5: Проверим корни на принадлежность ОДЗ**
Оба корня удовлетворяют ОДЗ: $\dfrac{4}{7} \neq 0$ и $\neq 0.5$, $-\dfrac{1}{2} \neq 0$ и $\neq 0.5$

**Ответ:** $x = \dfrac{4}{7}$, $x = -\dfrac{1}{2}$

</LessonSection>
