

<Section id="quadratic-function-definition" title="Основные свойства квадратичной функции и её графика">

<Def title="Определение квадратичной функции">

Квадратичной функцией называется функция вида

$$
y = ax^2 + bx + c
$$

где

* a, b, c — коэффициенты, причём a ≠ 0
* Графиком квадратичной функции является парабола

</Def>

</Section>

<Section id="parabola-direction" title="Направление ветвей параболы">

<Def title="Направление ветвей параболы зависит от коэффициента a">

Если a > 0

![](https://i.postimg.cc/fbnB2LNL/Вершина_Параболы.png)

* Ветви параболы направлены вверх
* Парабола имеет минимум в вершине
* Множество значений: E(f) = [y_в, +∞)

Если a < 0

![](https://i.postimg.cc/g2RKZW6K/Вершина_Параболы_a___0.png)

* Ветви параболы направлены вниз
* Парабола имеет максимум в вершине
* Множество значений: E(f) = (-∞, y_в]

</Def>

</Section>

<Section id="zeros-and-discriminant" title="Нули функции и дискриминант">

<Def title="Нули функции">

Нули функции — точки пересечения графика с осью OX (значения x, при которых y = 0). Определяются через дискриминант

$$
D = b^2 - 4ac
$$

D > 0 — два различных нуля

![](https://i.postimg.cc/4yqzYL44/Нули_функции.png)

* Парабола пересекает ось OX в двух точках: x₁, x₂
* y = a(x - x₁)(x - x₂)

D = 0 — один нуль (касание)

![](https://i.postimg.cc/K8nnYD6z/Нули_функции_D=0.png)

* Парабола касается оси OX в одной точке: x₀
* y = a(x - x₀)²

D < 0 — нет нулей

![](https://i.postimg.cc/sxkhjMKh/Нули_функции_D___0.png)

* Парабола не пересекает ось OX
* Функция не раскладывается на линейные множители

</Def>

</Section>

<Section id="axis-and-vertex" title="Ось симметрии и вершина параболы">

<Def title="Ось симметрии и вершина">

Ось симметрии параболы — вертикальная прямая, проходящая через вершину

![](https://i.postimg.cc/yNL3JQnM/Ось_симметрии.png)

Формула оси симметрии

$$
x_в = -\frac{b}{2a}
$$

Координаты вершины параболы

$$
x_в = -\frac{b}{2a}, \quad y_в = f(x_в) = -\frac{D}{4a}
$$

Функция в виде через вершину

$$
y = a(x - m)^2 + n
$$

где (m, n) — координаты вершины.

</Def>

</Section>

<Section id="graph-construction" title="Построение графика квадратичной функции">

<Def title="Алгоритм построения параболы">

1. Найти вершину и ось симметрии

Вычисляем координаты вершины:

$$
x_в = -\frac{b}{2a}, \quad y_в = f(x_в)
$$

![](https://i.postimg.cc/PxNK3Fqy/Построение_1.png)

2. Найти нули функции

Решаем уравнение ax² + bx + c = 0

Находим точки пересечения с осью OX (если они существуют)

![](https://i.postimg.cc/3r4n26Vk/Построение_2.png)

3. Найти точку пересечения с OY и симметричные точки

Точка пересечения с осью OY: (0, c)

Используем симметрию параболы для нахождения дополнительных точек

![](https://i.postimg.cc/mgRjfrD5/Построение_3-4.png)

</Def>

</Section>

<Section id="monotonicity" title="Монотонность квадратичной функции">

<Def title="Промежутки монотонности">

![](https://i.postimg.cc/hv1J9Kqw/Монотонность.png)

При a > 0

* Убывает на промежутке: (-∞, x_в]
* Возрастает на промежутке: [x_в, +∞)

При a < 0

* Возрастает на промежутке: (-∞, x_в]
* Убывает на промежутке: [x_в, +∞)

</Def>

</Section>

<Section id="sign-constancy" title="Промежутки знакопостоянства квадратичной функции">

<Def title="Промежутки знакопостоянства">

Промежутками знакопостоянства называются интервалы значений аргумента x, на которых функция сохраняет постоянный знак (положительный или отрицательный).

Знакопостоянство зависит от

1. Знака коэффициента a
2. Значения дискриминанта D
3. Расположения нулей функции (если они существуют)

</Def>

</Section>

<Section id="sign-constancy-a-positive" title="Знакопостоянство при a > 0">

<Def title="Случай 1: a > 0, D > 0">

![](https://i.postimg.cc/rsfVVsTr/Знакопостоянство.png)

* Функция имеет два различных нуля: x₁ и x₂ (x₁ < x₂)
* Положительна на: (-∞, x₁) ∪ (x₂, +∞)
* Отрицательна на: (x₁, x₂)
* Равна нулю в: x = x₁ и x = x₂

</Def>

<Def title="Случай 2: a > 0, D = 0">

![](https://i.postimg.cc/HW9jvWqV/Знакопостоянство_a___0.png)

* Функция имеет один нуль: x₀
* Положительна на: (-∞, x₀) ∪ (x₀, +∞)
* Отрицательных значений нет
* Равна нулю только в: x = x₀

</Def>

<Def title="Случай 3: a > 0, D < 0">

![](https://i.postimg.cc/BvPXmT0c/Знакопостоянство_a___0.png)

* Функция не имеет нулей
* Положительна на: (-∞, +∞)
* Отрицательных значений нет
* Нигде не равна нулю

</Def>

<Def title="Общее правило для a > 0">

При a > 0 парабола направлена ветвями вверх. Функция принимает положительные значения вне интервала между нулями (если они есть), и отрицательные значения между нулями. Если нулей нет, функция всегда положительна.

</Def>

</Section>

<Section id="sign-constancy-a-negative" title="Знакопостоянство при a < 0">

<Def title="Случай 1: a < 0, D > 0">

![](https://i.postimg.cc/TPjX21qG/изображение.png)

* Функция имеет два различных нуля: x₁ и x₂ (x₁ < x₂)
* Отрицательна на: (-∞, x₁) ∪ (x₂, +∞)
* Положительна на: (x₁, x₂)
* Равна нулю в: x = x₁ и x = x₂

</Def>

<Def title="Случай 2: a < 0, D = 0">

![](https://i.postimg.cc/SxMh5Dnj/изображение.png)

* Функция имеет один нуль: x₀
* Отрицательна на: (-∞, x₀) ∪ (x₀, +∞)
* Положительных значений нет
* Равна нулю только в: x = x₀

</Def>

<Def title="Случай 3: a < 0, D < 0">

![](https://i.postimg.cc/4y2GC0vd/изображение.png)

* Функция не имеет нулей
* Отрицательна на: (-∞, +∞)
* Положительных значений нет
* Нигде не равна нулю

</Def>

<Def title="Общее правило для a < 0">

При a < 0 парабола направлена ветвями вниз. Функция принимает отрицательные значения вне интервала между нулями (если они есть), и положительные значения между нулями. Если нулей нет, функция всегда отрицательна.

</Def>

</Section>

<Section id="sign-constancy-table" title="Таблица знакопостоянства квадратичной функции">

<Def title="Коэффициент a > 0">

| Дискриминант D | Нули функции | y > 0 | y < 0 |
| :--- | :--- | :--- | :--- |
| D > 0 | Два нуля: x₁, x₂ (x₁ < x₂) | (-∞, x₁) ∪ (x₂, +∞) | (x₁, x₂) |
| D = 0 | Один нуль: x₀ | (-∞, x₀) ∪ (x₀, +∞) | ∅ (нет) |
| D < 0 | Нет нулей | (-∞, +∞) | ∅ (нет) |

</Def>

<Def title="Коэффициент a < 0">

| Дискриминант D | Нули функции | y > 0 | y < 0 |
| :--- | :--- | :--- | :--- |
| D > 0 | Два нуля: x₁, x₂ (x₁ < x₂) | (x₁, x₂) | (-∞, x₁) ∪ (x₂, +∞) |
| D = 0 | Один нуль: x₀ | ∅ (нет) | (-∞, x₀) ∪ (x₀, +∞) |
| D < 0 | Нет нулей | ∅ (нет) | (-∞, +∞) |

</Def>

<Def title="Запомните">

1. При a > 0: функция положительна снаружи нулей и отрицательна между нулями
2. При a < 0: функция отрицательна снаружи нулей и положительна между нулями
3. Если нулей нет (D < 0): знак функции совпадает со знаком коэффициента a на всей числовой прямой

</Def>

</Section>

<Section id="function-analysis-examples" title="Анализ конкретных функций">

<Ex title="Функция 1: f(x) = -(x - 1,2)² + 25">

**Условие:** Исследовать функцию f(x) = -(x - 1,2)² + 25 и найти её свойства.

Форма записи: Вид через координаты вершины: y = a(x - m)² + n

Область определения: D(f) = ℝ

*Квадратичная функция определена для всех действительных чисел*

Коэффициент a: a = -1 < 0

*Ветви параболы направлены вниз*

Координаты вершины: m = 1,2, n = 25

Вершина: (1,2; 25)

Ось симметрии: x = 1,2

Множество значений: E(f) = (-∞, 25]

*При a < 0 функция имеет максимум в вершине*

Нули функции

$$
-(x - 1,2)^2 + 25 = 0
$$

$$
(x - 1,2)^2 = 25
$$

$$
x - 1,2 = \pm5
$$

$$
x_1 = 1,2 - 5 = -3,8
$$

$$
x_2 = 1,2 + 5 = 6,2
$$

*D > 0 — два различных нуля*

Промежутки монотонности

* Возрастает: (-∞, 1,2]
* Убывает: [1,2, +∞)

</Ex>

<Ex title="Функция 2: h(x) = 2(x - 1)(x + 4)">

**Условие:** Исследовать функцию h(x) = 2(x - 1)(x + 4) и найти её свойства.

Форма записи: Факторизованный вид: y = a(x - x₁)(x - x₂)

Область определения: D(h) = ℝ

Коэффициент a: a = 2 > 0

*Ветви параболы направлены вверх*

Нули функции: Из факторизованного вида: x₁ = 1, x₂ = -4

*D > 0 — два различных нуля*

Координаты вершины

$$
x_в = \frac{1 + (-4)}{2} = \frac{-3}{2} = -1,5
$$

$$
y_в = h(-1,5) = 2 \times (-2,5) \times 2,5 = -12,5
$$

Вершина: (-1,5; -12,5)

*Находим среднее арифметическое нулей*

Множество значений: E(h) = [-12,5, +∞)

*При a > 0 функция имеет минимум в вершине*

Промежутки монотонности

* Убывает: (-∞, -1,5]
* Возрастает: [-1,5, +∞)

</Ex>

<Ex title="Функция 3: p(x) = -2x² + 4x - 2">

**Условие:** Исследовать функцию p(x) = -2x² + 4x - 2 и найти её свойства.

Форма записи: Общий вид: y = ax² + bx + c

Область определения: D(p) = ℝ

Коэффициенты: a = -2 < 0, b = 4, c = -2

*Ветви параболы направлены вниз*

Дискриминант

$$
D = b^2 - 4ac = 4^2 - 4 \times (-2) \times (-2) = 16 - 16 = 0
$$

*D = 0 — один нуль (касание оси OX)*

Нуль функции

$$
x_0 = -\frac{b}{2a} = -\frac{4}{2 \times (-2)} = -\frac{4}{-4} = 1
$$

Нуль: x = 1

Координаты вершины

$$
x_в = 1 \text{ (совпадает с нулём)}, \quad y_в = p(1) = -2 \times 1^2 + 4 \times 1 - 2 = -2 + 4 - 2 = 0
$$

Вершина: (1; 0)

Ось симметрии: x = 1

Множество значений: E(p) = (-∞, 0]

Промежутки монотонности

* Возрастает: (-∞, 1]
* Убывает: [1, +∞)

Факторизованный вид: p(x) = -2(x - 1)²

*При D = 0 функция имеет вид y = a(x - x₀)²*

</Ex>
