

<Section id="quadratic-equation-definition" title="Что такое квадратное уравнение">

<Def title="Определение квадратного уравнения">
Квадратным уравнением называется уравнение вида

$$
ax^2 + bx + c = 0
$$

где
* x — переменная (неизвестное)
* a, b, c — некоторые числа (коэффициенты)
* a ≠ 0 (если a = 0, уравнение становится линейным)

Пояснение: Квадратное уравнение называется так потому, что наивысшая степень переменной x — вторая (x²). Число a называется первым коэффициентом или коэффициентом при x², число b — вторым коэффициентом или коэффициентом при x, а число c — свободным членом.
</Def>

</Section>

<Section id="incomplete-quadratic-equations" title="Неполные квадратные уравнения">

<Def title="Определение неполного квадратного уравнения">
Неполным квадратным уравнением называется квадратное уравнение, в котором один или два коэффициента (кроме a) равны нулю.

Три вида неполных квадратных уравнений:

Вид 1: Уравнение без свободного члена (c = 0)

$$
ax^2 + bx = 0
$$

где a ≠ 0 и b ≠ 0.

Вид 2: Уравнение без линейного члена (b = 0)

$$
ax^2 + c = 0
$$

где a ≠ 0 и c ≠ 0.

Вид 3: Уравнение только с квадратом переменной

$$
ax^2 = 0
$$

где a ≠ 0 (b = 0 и c = 0).
</Def>

</Section>

<Section id="solving-incomplete-quadratics" title="Решение неполных квадратных уравнений">

<Ex title="Пример 1: Решение уравнения вида ax² = 0">
Решите уравнение

$$
-2x^2 = 0
$$

Решение

1. Разделим обе части уравнения на -2

$$
\frac{-2x^2}{-2} = \frac{0}{-2}
$$

2. Упрощаем

$$
x^2 = 0
$$

3. Извлекаем квадратный корень

$$
x = 0
$$

Ответ

$$
x = 0
$$
</Ex>

<Ex title="Пример 2: Решение уравнения вида ax² + bx = 0">
Решите уравнение

$$
5x^2 + 2x = 0
$$

Решение

1. Вынесем общий множитель x за скобки

$$
x(5x + 2) = 0
$$

2. Произведение равно нулю, когда хотя бы один из множителей равен нулю

$$
x = 0 \quad \text{или} \quad 5x + 2 = 0
$$

3. Решаем второе уравнение

$$
5x = -2
$$

$$
x = -\frac{2}{5}
$$

Ответ

$$
x_1 = 0, \quad x_2 = -\frac{2}{5}
$$
</Ex>

<Ex title="Пример 3: Решение уравнения вида ax² + c = 0 (c > 0)">
Решите уравнение

$$
x^2 - 3 = 0
$$

Решение

1. Перенесём свободный член в правую часть

$$
x^2 = 3
$$

2. Извлечём квадратный корень из обеих частей

$$
x = \pm\sqrt{3}
$$

Ответ

$$
x = \pm\sqrt{3}
$$
</Ex>

<Ex title="Пример 4: Решение уравнения вида ax² + c = 0 (c < 0, a < 0)">
Решите уравнение

$$
-x^2 - 1 = 0
$$

Решение

1. Перенесём свободный член в правую часть

$$
-x^2 = 1
$$

2. Разделим обе части на -1

$$
x^2 = -1
$$

3. Квадрат действительного числа не может быть отрицательным

Ответ: уравнение не имеет действительных корней
</Ex>

</Section>

<Section id="complete-quadratic-definition" title="Что такое полное квадратное уравнение">

<Def title="Определение полного квадратного уравнения">
Полным квадратным уравнением называется уравнение вида

$$
ax^2 + bx + c = 0
$$

где
* a, b, c — коэффициенты, причём a ≠ 0
* Все три коэффициента отличны от нуля: a ≠ 0, b ≠ 0, c ≠ 0

Пояснение: В отличие от неполных квадратных уравнений, в полных квадратных уравнениях присутствуют все три члена: квадратичный член (ax²), линейный член (bx) и свободный член (c).
</Def>

</Section>

<Section id="discriminant" title="Дискриминант квадратного уравнения">

<Def title="Определение дискриминанта">
Дискриминантом квадратного уравнения называется выражение

$$
D = b^2 - 4ac
$$

Дискриминант (от латинского discriminare — различать) определяет количество и тип корней квадратного уравнения.

Правило определения корней по дискриминанту

| Дискриминант D | Количество корней | Тип корней |
| :--- | :--- | :--- |
| D > 0 | Два различных корня | Действительные, различные |
| D = 0 | Один корень (два одинаковых) | Действительные, равные |
| D < 0 | Нет действительных корней | Комплексные (в школьном курсе — нет корней) |
</Def>

</Section>

<Section id="quadratic-formula" title="Формула корней квадратного уравнения">

<Def title="Формула корней">
Формула для нахождения корней квадратного уравнения

$$
x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}
$$

где D = b² - 4ac — дискриминант.

Пояснение: Знак ± означает, что нужно вычислить два значения: одно со знаком плюс, другое со знаком минус. Формула работает для любого квадратного уравнения с действительными коэффициентами.
</Def>

</Section>

<Section id="complete-quadratic-examples" title="Примеры решения полных квадратных уравнений">

<Ex title="Пример 1: Уравнение с D > 0 (два различных корня)">
Решите уравнение

$$
2x^2 - 5x + 2 = 0
$$

Решение

1. Определяем коэффициенты: a = 2, b = -5, c = 2
2. Вычисляем дискриминант

$$
D = (-5)^2 - 4 \cdot 2 \cdot 2 = 25 - 16 = 9
$$

3. Так как D = 9 > 0, уравнение имеет два различных корня
4. Находим корни по формуле

$$
x_{1,2} = \frac{5 \pm \sqrt{9}}{4} = \frac{5 \pm 3}{4}
$$

5. Вычисляем два корня

$$
x_1 = \frac{5 + 3}{4} = 2
$$

$$
x_2 = \frac{5 - 3}{4} = \frac{1}{2}
$$

Ответ

$$
x_1 = 2, \quad x_2 = \frac{1}{2}
$$
</Ex>

<Ex title="Пример 2: Уравнение с D = 0 (один корень)">
Решите уравнение

$$
x^2 - 6x + 9 = 0
$$

Решение

1. Определяем коэффициенты: a = 1, b = -6, c = 9
2. Вычисляем дискриминант

$$
D = (-6)^2 - 4 \cdot 1 \cdot 9 = 36 - 36 = 0
$$

3. Так как D = 0, уравнение имеет один корень (два одинаковых)
4. Находим корень по формуле

$$
x = \frac{6}{2 \cdot 1} = \frac{6}{2} = 3
$$

Ответ

$$
x = 3
$$
</Ex>

<Ex title="Пример 3: Уравнение с D < 0 (нет действительных корней)">
Решите уравнение

$$
3x^2 - 4x + 5 = 0
$$

Решение

1. Определяем коэффициенты: a = 3, b = -4, c = 5
2. Вычисляем дискриминант

$$
D = (-4)^2 - 4 \cdot 3 \cdot 5 = 16 - 60 = -44
$$

3. Так как D = -44 < 0, уравнение не имеет действительных корней

Ответ: уравнение не имеет действительных корней
</Ex>

</Section>

<Section id="vieta-theorem-reduced" title="Теорема Виета для приведённого квадратного уравнения">

<Def title="Теорема Виета (приведённая форма)">
Теорема Виета устанавливает связь между корнями квадратного уравнения и его коэффициентами.

Для приведённого квадратного уравнения вида

$$
x^2 + px + q = 0
$$

верны следующие соотношения

$$
x_1 + x_2 = -p
$$

$$
x_1 \cdot x_2 = q
$$

где x₁ и x₂ — корни уравнения.

Пояснение: Теорема Виета позволяет
* Находить корни уравнения подбором
* Проверять правильность найденных корней
* Находить выражения, составленные из корней, без вычисления самих корней
</Def>

</Section>

<Section id="vieta-theorem-general" title="Теорема Виета для общего квадратного уравнения">

<Def title="Теорема Виета (общая форма)">
Для общего квадратного уравнения вида

$$
ax^2 + bx + c = 0 \quad (a \neq 0)
$$

соотношения Виета имеют вид

$$
x_1 + x_2 = -\frac{b}{a}
$$

$$
x_1 \cdot x_2 = \frac{c}{a}
$$
</Def>

</Section>

<Section id="vieta-examples" title="Примеры решения уравнений с помощью теоремы Виета">

<Ex title="Пример 1: Решение уравнения подбором">
Решите уравнение

$$
x^2 - 4x + 3 = 0
$$

Решение с использованием теоремы Виета

1. Запишем условия теоремы Виета для данного уравнения

$$
x_1 + x_2 = 4 \quad \text{(так как } -p = -(-4) = 4\text{)}
$$

$$
x_1 \cdot x_2 = 3
$$

2. Подбираем пару чисел, сумма которых равна 4, а произведение равно 3

   Проверяем возможные варианты
   * 1 + 3 = 4 и 1 × 3 = 3 ✓
   * 2 + 2 = 4, но 2 × 2 = 4 ≠ 3
   * 4 + 0 = 4, но 4 × 0 = 0 ≠ 3

Ответ

$$
x_1 = 1, \quad x_2 = 3
$$
</Ex>

<Ex title="Пример 2: Решение уравнения подбором">
Решите уравнение

$$
x^2 + 7x + 10 = 0
$$

Решение с использованием теоремы Виета

1. Запишем условия теоремы Виета для данного уравнения

$$
x_1 + x_2 = -7
$$

$$
x_1 \cdot x_2 = 10
$$

2. Подбираем пару чисел, сумма которых равна -7, а произведение равно 10

   Оба корня должны быть отрицательными (так как сумма отрицательна, а произведение положительно)

   Проверяем возможные варианты
   * (-2) + (-5) = -7 и (-2) × (-5) = 10 ✓
   * (-1) + (-6) = -7, но (-1) × (-6) = 6 ≠ 10
   * (-3) + (-4) = -7, но (-3) × (-4) = 12 ≠ 10

Ответ

$$
x_1 = -2, \quad x_2 = -5
$$
</Ex>

</Section>

<Section id="expressions-from-roots" title="Выражения, составленные из корней">

<Def title="Полезные формулы">
С помощью теоремы Виета можно вычислять различные выражения, составленные из корней, не находя сами корни.

Основные полезные формулы

1. Сумма квадратов корней

$$
x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2
$$

2. Сумма кубов корней

$$
x_1^3 + x_2^3 = (x_1 + x_2)^3 - 3x_1x_2(x_1 + x_2)
$$

3. Выражение вида x₁²x₂ + x₁x₂²

$$
x_1^2x_2 + x_1x_2^2 = x_1x_2(x_1 + x_2)
$$
</Def>

<Ex title="Пример вычисления выражений без нахождения корней">
Задача
Известно, что x₁ и x₂ — корни уравнения x² + 7x - 12 = 0.
Не решая уравнения, найдите значение выражений

Решение

1. Сначала определим по теореме Виета

$$
x_1 + x_2 = -7
$$

$$
x_1 \cdot x_2 = -12
$$

2. а) Найдём (x₁ + x₂)²

$$
(x_1 + x_2)^2 = (-7)^2 = 49
$$

3. б) Найдём x₁²x₂ + x₁x₂²

$$
x_1^2x_2 + x_1x_2^2 = x_1x_2(x_1 + x_2) = (-12) \cdot (-7) = 84
$$

4. в) Найдём x₁² + x₂²

$$
x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2 = (-7)^2 - 2 \cdot (-12) = 49 + 24 = 73
$$

Ответ: а) 49, б) 84, в) 73
</Ex>

</Section>

<Section id="inverse-vieta" title="Обратная теорема Виета">

<Def title="Обратная теорема Виета">
Обратная теорема Виета: Если числа x₁ и x₂ таковы, что

$$
x_1 + x_2 = -p
$$

$$
x_1 \cdot x_2 = q
$$

то эти числа являются корнями квадратного уравнения

$$
x^2 + px + q = 0
$$
</Def>

<Ex title="Пример использования обратной теоремы">
Составьте квадратное уравнение, корни которого равны 2 и -5.

Решение

1. Находим сумму корней

$$
2 + (-5) = -3
$$

2. Находим произведение корней

$$
2 \times (-5) = -10
$$

3. По обратной теореме Виета, уравнение имеет вид

$$
x^2 + 3x - 10 = 0
$$

Ответ

$$
x^2 + 3x - 10 = 0
$$
</Ex>

</Section>

<Section id="factoring-quadratic-trinomial" title="Теорема о разложении квадратного трехчлена на множители">

<Def title="Квадратный трехчлен и его разложение">
Квадратный трехчлен — это выражение вида

$$
ax^2 + bx + c
$$

где a ≠ 0.

Теорема: Если квадратное уравнение ax² + bx + c = 0 имеет корни x₁ и x₂, то квадратный трехчлен можно разложить на множители

$$
ax^2 + bx + c = a(x - x_1)(x - x_2)
$$

Пояснение: Эта теорема позволяет разложить квадратный трехчлен на линейные множители, если соответствующее квадратное уравнение имеет корни. Если уравнение не имеет корней (D < 0), то разложение на линейные множители с действительными коэффициентами невозможно.
</Def>

<Def title="Алгоритм разложения">
Шаги разложения

1. Найти корни квадратного уравнения ax² + bx + c = 0
2. Если уравнение имеет два корня x₁ и x₂, то разложить по формуле

$$
a(x - x_1)(x - x_2)
$$

3. Если уравнение имеет один корень x₁ (D = 0), то

$$
a(x - x_1)^2
$$

4. Если уравнение не имеет корней (D < 0), то разложение на линейные множители с действительными коэффициентами невозможно
</Def>

</Section>

<Section id="factoring-examples" title="Примеры разложения квадратных трехчленов на множители">

<Ex title="Пример 1: 4x² - 8x + 3">
Решение

1. Составляем квадратное уравнение

$$
4x^2 - 8x + 3 = 0
$$

2. Находим дискриминант

$$
D = (-8)^2 - 4 \cdot 4 \cdot 3 = 64 - 48 = 16
$$

3. Находим корни

$$
x_{1,2} = \frac{8 \pm \sqrt{16}}{8} = \frac{8 \pm 4}{8}
$$

$$
x_1 = \frac{12}{8} = \frac{3}{2}, \quad x_2 = \frac{4}{8} = \frac{1}{2}
$$

4. Разлагаем на множители

$$
4x^2 - 8x + 3 = 4\left(x - \frac{3}{2}\right)\left(x - \frac{1}{2}\right)
$$

5. Можно записать без дробей

$$
4x^2 - 8x + 3 = (2x - 3)(2x - 1)
$$

Ответ

$$
(2x - 3)(2x - 1)
$$
</Ex>

<Ex title="Пример 2: 81t² - 36t + 4">
Решение

1. Составляем уравнение

$$
81t^2 - 36t + 4 = 0
$$

2. Находим дискриминант

$$
D = (-36)^2 - 4 \cdot 81 \cdot 4 = 1296 - 1296 = 0
$$

3. Так как D = 0, уравнение имеет один корень

$$
t = \frac{36}{2 \cdot 81} = \frac{36}{162} = \frac{2}{9}
$$

4. Разлагаем на множители

$$
81t^2 - 36t + 4 = 81\left(t - \frac{2}{9}\right)^2
$$

5. Можно записать как полный квадрат

$$
81t^2 - 36t + 4 = (9t - 2)^2
$$

Ответ

$$
(9t - 2)^2
$$
</Ex>

<Ex title="Пример 3: 8t² - 6t + 3">
Решение

1. Составляем уравнение

$$
8t^2 - 6t + 3 = 0
$$

2. Находим дискриминант

$$
D = (-6)^2 - 4 \cdot 8 \cdot 3 = 36 - 96 = -60
$$

3. Так как D = -60 < 0, квадратное уравнение не имеет действительных корней

4. Следовательно, квадратный трехчлен нельзя разложить на линейные множители с действительными коэффициентами

Ответ: разложение на линейные множители с действительными коэффициентами невозможно
</Ex>

</Section>

<Section id="biquadratic-equations" title="Биквадратные уравнения">

<Def title="Определение биквадратного уравнения">
Биквадратным уравнением называется уравнение вида

$$
ax^4 + bx^2 + c = 0
$$

где a ≠ 0.

Пояснение: Биквадратное уравнение содержит только чётные степени переменной. Его можно свести к квадратному уравнению с помощью замены переменной.
</Def>

<Def title="Метод решения биквадратных уравнений">
Алгоритм решения

1. Сделать замену переменной: t = x², при этом t ≥ 0
2. Получить квадратное уравнение: at² + bt + c = 0
3. Решить полученное квадратное уравнение
4. Для каждого неотрицательного корня t ≥ 0 решить уравнение x² = t
5. Записать все действительные корни исходного уравнения
</Def>

<Ex title="Пример решения биквадратного уравнения">
Решите уравнение

$$
x^4 - 10x^2 + 9 = 0
$$

Решение

1. Делаем замену: t = x², тогда t ≥ 0
2. Получаем квадратное уравнение

$$
t^2 - 10t + 9 = 0
$$

3. Решаем квадратное уравнение

$$
D = 100 - 36 = 64
$$

$$
t_{1,2} = \frac{10 \pm \sqrt{64}}{2} = \frac{10 \pm 8}{2}
$$

4. Находим значения t

$$
t_1 = \frac{18}{2} = 9, \quad t_2 = \frac{2}{2} = 1
$$

5. Возвращаемся к переменной x

   * Если t = 9, то x² = 9 ⇒ x = ±3
   * Если t = 1, то x² = 1 ⇒ x = ±1

Ответ

$$
x_1 = -3, \quad x_2 = -1, \quad x_3 = 1, \quad x_4 = 3
$$
</Ex>

</Section>

<Section id="substitution-method" title="Уравнения, сводящиеся к квадратным с помощью замены переменной">

<Ex title="Пример: (x² - 4x)² + 7(x² - 4x) + 12 = 0" isHard>
Решение

1. Замечаем повторяющееся выражение x² - 4x
2. Делаем замену: t = x² - 4x
3. Получаем квадратное уравнение

$$
t^2 + 7t + 12 = 0
$$

4. Решаем квадратное уравнение

$$
D = 49 - 48 = 1
$$

$$
t_{1,2} = \frac{-7 \pm \sqrt{1}}{2} = \frac{-7 \pm 1}{2}
$$

5. Находим значения t

$$
t_1 = \frac{-6}{2} = -3, \quad t_2 = \frac{-8}{2} = -4
$$

6. Возвращаемся к переменной x

   Случай 1: x² - 4x = -3

$$
x^2 - 4x + 3 = 0
$$

   Решаем: x₁ = 1, x₂ = 3

   Случай 2: x² - 4x = -4

$$
x^2 - 4x + 4 = 0
$$

   Решаем: (x - 2)² = 0 ⇒ x₃ = 2

Ответ

$$
x_1 = 1, \quad x_2 = 2, \quad x_3 = 3
$$
</Ex>

</Section>

<Section id="absolute-value-equations" title="Уравнения с модулем, сводящиеся к квадратным">

<Ex title="Пример: x² - 3|x| + 2 = 0" isHard>
Решение

Рассмотрим два случая в зависимости от знака x

Случай 1: x ≥ 0, тогда |x| = x

Уравнение

$$
x^2 - 3x + 2 = 0
$$

Решение

$$
D = 9 - 8 = 1
$$

$$
x = \frac{3 \pm 1}{2}
$$

$$
x_1 = 1, \quad x_2 = 2 \quad \text{(оба удовлетворяют условию } x \geq 0\text{)}
$$

Случай 2: x < 0, тогда |x| = -x

Уравнение

$$
x^2 + 3x + 2 = 0
$$

Решение

$$
D = 9 - 8 = 1
$$

$$
x = \frac{-3 \pm 1}{2}
$$

$$
x_3 = -1, \quad x_4 = -2 \quad \text{(оба удовлетворяют условию } x < 0\text{)}
$$

Ответ

$$
x_1 = -2, \quad x_2 = -1, \quad x_3 = 1, \quad x_4 = 2
$$

Альтернативный метод: Замена t = |x|, t ≥ 0. Тогда уравнение принимает вид t² - 3t + 2 = 0.
