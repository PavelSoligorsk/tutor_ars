<LessonSection title="Линейная функция">

<Def>
**Линейная функция** — функция вида $y = kx + b$, где:
- $x$ — независимая переменная (аргумент)
- $y$ — зависимая переменная (функция)
- $k$ — коэффициент наклона (угловой коэффициент)
- $b$ — свободный член (координата пересечения с осью Oy)
</Def>

<Def>
**Геометрический смысл коэффициентов:**
- $k$ — тангенс угла наклона прямой к положительному направлению оси Ox
- $b$ — ордината точки пересечения графика с осью Oy (при $x = 0$)
</Def>

<ImageContainer src="https://i.postimg.cc/wMkQ64F5/изображение.png" alt="4 варианта линейной функции в зависимости от знаков k и b" />
<Def>
**Влияние знаков k и b на график:**

1. **k > 0, b > 0:**
   - Прямая возрастает
   - Пересекает ось Oy выше начала координат

2. **k < 0, b > 0:**
   - Прямая убывает
   - Пересекает ось Oy выше начала координат

3. **k > 0, b < 0:**
   - Прямая возрастает
   - Пересекает ось Oy ниже начала координат

4. **k < 0, b < 0:**
   - Прямая убывает
   - Пересекает ось Oy ниже начала координат
</Def>

<ImageContainer src="https://i.postimg.cc/C1RjyLfw/изображение.png" alt="3 варианта взаимного расположения графиков линейных функций" />
<Def>
**Взаимное расположение графиков линейных функций:**

1. **Параллельные прямые:**
   - Условие: $k_1 = k_2$, $b_1 ≠ b_2$
   - Графики не пересекаются

2. **Пересекающиеся прямые:**
   - Условие: $k_1 ≠ k_2$
   - Графики пересекаются в одной точке


3. **Совпадающие прямые:**
   - Условие: $k_1 = k_2$, $b_1 = b_2$
   - Графики полностью совпадают
</Def>

<ImageContainer src="https://i.postimg.cc/bvHx0BcK/изображение.png" alt="Построение графиков линейных функций" />
<Def>
**Примеры построения графиков:**

1. **Построение прямой x = -1:**
   - Вертикальная прямая, параллельная оси Oy
   - Все точки имеют координату x = -1
   - Отмечена точка (-1, 3)

2. **Построение прямой y = 2x + 1:**
   - Находим две точки:
     * При x = -1: y = 2·(-1) + 1 = -1 → точка (-1, -1)
     * При x = 1: y = 2·1 + 1 = 3 → точка (1, 3)
   - Проводим прямую через эти точки

3. **Построение прямой y = 3:**
   - Горизонтальная прямая, параллельная оси Ox
   - Все точки имеют координату y = 3
   - Можно отметить точки: (-2, 3), (0, 3), (2, 3)
   - Проходит через точку (-1, 3) из первого примера


</Def>

<Def>
**Способы задания линейной функции:**
1. Аналитический: $y = kx + b$
2. Графический: прямая линия на координатной плоскости
3. Табличный: таблица значений x и y
4. Словесный: описание зависимости между величинами
</Def>
