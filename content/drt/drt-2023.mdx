### Задание 3 (1 балл) | Преобразование выражений

Найдите значение выражения:

$$\frac{(p^{1-\sqrt{10}})^{\sqrt{10}+1} \cdot p^{\sqrt{10}-2}}{p^{\sqrt{10}-3}}$$

при $p = \frac{1}{\sqrt[4]{19}}$.

<CheckTask
  id="task-3"
  type="choice"
  multiple='{false}'
  options='[
    "$\\frac{1}{\\sqrt[4]{19}}$",
    "$\\frac{1}{19}$",
    "$\\sqrt[4]{19}$",
    "$19$",
    "$\\frac{1}{\\sqrt{19}}$"
  ]'
  answer="$\\frac{1}{19}$"
  hint="
**Подсказка:**

Упростите выражение, используя свойства степеней:

$$(p^{1-\sqrt{10}})^{\sqrt{10}+1} = p^{(1-\sqrt{10})(\sqrt{10}+1)} = p^{1 - 10} = p^{-9}$$

$$\\frac{p^{-9} \\cdot p^{\\sqrt{10}-2}}{p^{\\sqrt{10}-3}} = p^{-9 + (\\sqrt{10}-2) - (\\sqrt{10}-3)} = p^{-9 + \\sqrt{10} - 2 - \\sqrt{10} + 3} = p^{-8}$$

При $p = \\frac{1}{\\sqrt[4]{19}}$:

$$p^{-8} = \\left(\\frac{1}{\\sqrt[4]{19}}\\right)^{-8} = (\\sqrt[4]{19})^8 = 19^2 = 361$$
  "
  explanation="
**Решение:**

**Шаг 1:** Упрощаем показатель степени в числителе

$$(p^{1-\sqrt{10}})^{\sqrt{10}+1} = p^{(1-\sqrt{10})(\sqrt{10}+1)}$$

$$(1-\sqrt{10})(\sqrt{10}+1) = 1 - 10 = -9$$

Получаем: $p^{-9}$

**Шаг 2:** Записываем всё выражение

$$\\frac{p^{-9} \\cdot p^{\\sqrt{10}-2}}{p^{\\sqrt{10}-3}} = p^{-9 + (\\sqrt{10}-2) - (\\sqrt{10}-3)}$$

**Шаг 3:** Упрощаем сумму показателей

$$-9 + \\sqrt{10} - 2 - \\sqrt{10} + 3 = -8$$

Выражение равно $p^{-8}$

**Шаг 4:** Подставляем $p = \\frac{1}{\\sqrt[4]{19}}$

$$p^{-8} = \\left(\\frac{1}{\\sqrt[4]{19}}\\right)^{-8} = (\\sqrt[4]{19})^8 = (19^{1/4})^8 = 19^2 = 361$$

**Ответ:** $361$
  "
